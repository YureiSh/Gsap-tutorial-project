# GSAP Learning Project

A small **tutorial** project built with **React** and **GSAP** to practice modern web animations. I also put it on to vercel domain:
https://gsap-tutorial-project.vercel.app

## 🚀 About

This project was created to explore and experiment with the GSAP animation library in a React environment. It includes examples of timeline animations, scroll-triggered effects, and text animations.
